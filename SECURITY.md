# Security Policy

## Reporting

Please report security issues privately:

```text
serkan@altuntas.dev
```

Do not open a public issue for vulnerabilities, secret exposure, unsafe model
artifact distribution, or data leakage.

## Scope

Security-sensitive areas include:

- CLI execution behavior;
- Git repository access;
- local model loading;
- release artifacts;
- data and generated-label provenance;
- accidental inclusion of private repositories or held-out evals.
